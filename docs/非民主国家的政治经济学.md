## 课程描述

非民主政权一直以来都是最古老和最常见的政治治理形式。世界上绝大多数人口一直以来都难以获得政治（和经济）制度。然而，直到最近，政治经济学领域绝大多数的学术努力都致力于研究成熟的民主国家。

本课程介绍非民主国家的政治经济学主题。我们将研究非民主政权的稳定力量和变革力量。关于稳定力量，我们将涵盖诸如镇压、仇恨、国家强制、信息控制、腐败、收买以及政治信任和规范等主题。关于变革力量，我们将涵盖诸如民主化、集体行动、冲突、国家建设、改革和私有化、国家能力、官僚机构以及制度对经济发展和创新的作用等主题。

我们将从理论和实证两个方面探讨这些主题。学生将接触到政治经济学中的各种常用模型，以及实证方法的最新进展，如自然实验、田野实验、实验室实验、田野实验室实验和文本分析。

## 课程要求和评分

1. **课堂参与 (25%)：** 学生应积极参与课堂讨论，这是课程的核心部分。在每节课开始之前，学生需要撰写一篇关于该课程指定阅读材料的简短摘要，并将其上传到课程网站。至少，目标应该是提出你在阅读摘要中提出的任何问题。

2. **“模拟”审稿报告 (30%)：** 学生需要选择两篇政治经济学领域的未发表论文，并为每篇论文撰写一份审稿报告。审稿报告应包括论文的简短摘要、对其优缺点的评估以及改进建议。以下是一些关于如何成为审稿人的有用文章：
    * Preparing a Referee Report: Guidelines and Perspectives (准备审稿报告：指南和视角) [here](here)
    * How to Write an Effective Referee Report (如何撰写有效的审稿报告) [here](here)

3. **研究计划书 (10%)：** 两页的研究论文提案。这应该清楚地陈述你的研究问题。如果拟议的论文是实证性的，它还应明确报告回归方程和数据来源。如果拟议的论文是理论性的，则应提供对模型基础的简要描述。研究计划书应在期中提交。提交后，学生将安排与导师的办公时间会议，讨论研究计划书。

4. **期末演讲 (35%)：** 学生需要准备一个 20 分钟的演讲，内容是与课程主题相关的新项目，定义广泛。这应该遵循研究计划书。演讲将在学期末进行。虽然不要求提交期末论文，但鼓励学生根据演讲内容撰写论文。

## 生成式人工智能政策

本课程允许学生探索使用生成式人工智能 (GAI) 工具（如 ChatGPT）来收集或清理数据、编码以及研究工作流程中的所有其他任务。这些工具不得用于撰写审稿报告和研究计划书。评估提交的任何 GAI 输出的有效性和适用性是每个学生的责任；你承担最终责任。违反此政策将被视为学术不端行为。我们提请你注意，哈佛大学的不同课程可能会实施不同的人工智能政策，学生有责任遵守每门课程的期望。
